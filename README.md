# Labs-minimal-api-net-core

## Commands

### Create project

```bash
dotnet new web -n Labs.MinimalApi
```

> Donde el flag -n es el nombre del proyecto

### Run project

```bash
dotnet run --project Microservice/
```

### Add package

Add swagger package

```bash
dotnet add Microservice/ Swashbuckle.AspNetCore.Annotations
```
